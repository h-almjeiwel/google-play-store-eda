# Exploratory Data Analysis of Google Play Store Apps

## Project Objective
To explore and analyze the Google Play Store dataset through data cleaning, category distribution, install volumes, pricing, and free vs. paid patterns.

## Key Insights

- The Family category has the highest number of apps, while the Game category leads in total installs.
- Free applications significantly outnumber paid apps across major categories.
- Most paid apps are priced between $1 and $5.
- Gaming, Communication, and Tools generate the highest total install volumes.
- Apps priced above $50 are rare within the dataset.
- The dataset was refined from 10,841 raw records to 8,196 clean observations.

## Technical Approach
- Data Cleaning: Removing duplicates and handling missing values.
- Data Formatting: Adjusting column data types to enable accurate analysis.
- Data Preparation: Standardizing column names and cleaning and organizing the dataset.
- Exploratory Analysis: Grouping and aggregating data to identify patterns across categories.
- Visualization: Creating charts to show category performance and price distribution

## Dataset
10,841 raw records refined to 8,196 clean observations.

## Tools
Python (Pandas, Seaborn, Matplotlib)
